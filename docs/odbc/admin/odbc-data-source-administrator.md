# <a name="odbc-data-source-administrator"></a>Administrateur des sources de données ODBC
L®'administrateur de source de données de Microsoft ® ODBC gère les pilotes de bases de données et les sources de données. Cette application est située dans le panneau de contrôle Windows sous **outils administratifs**. À partir de Windows 8, l’icône est nommée **ODBC Data Sources**, et sur les systèmes d’exploitation 64 bits il ya une version 32 bits et 64 bits.  
  
 Pour obtenir de l’information sur les procédures détaillées de l’administrateur de l’ODBC, ouvrez la boîte de dialogue [de l’administrateur de source de données ODBC](https://msdn.microsoft.com/eea94d94-f53b-4289-ae75-9ccccde15333) et cliquez sur **Help**.  
  
 Vous pouvez utiliser les commandes PowerShell pour gérer les pilotes et les sources de données. Pour plus d’informations sur ces commandes PowerShell, voir [Windows Data Access Components PowerShell commandes](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx).
