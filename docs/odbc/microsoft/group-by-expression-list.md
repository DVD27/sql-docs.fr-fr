# <a name="group-by-expression-list"></a>GROUP BY, liste d’expressions
GROUP BY prend en charge une liste d’expressions, ainsi qu’un nom de colonne. La liste de sélection d’une instruction SELECT qui contient une clause GROUP BY ne peut inclure que des expressions de la clause GROUP BY ou des fonctions set.
