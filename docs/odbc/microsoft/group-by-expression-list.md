# <a name="group-by-expression-list"></a>GROUP BY, liste d’expressions
GROUP BY prend en charge une liste d’expressions ainsi qu’un nom de colonne. La liste sélectionnée d’une déclaration SELECT qui a une clause GROUPE BY ne peut inclure que des expressions de la clause GROUPE BY ou des fonctions définies.
