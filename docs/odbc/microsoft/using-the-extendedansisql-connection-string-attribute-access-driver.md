# <a name="using-the-extendedansisql-connection-string-attribute-access-driver"></a>Utilisation de l’attribut de chaîne de connexion ExtendedAnsiSQL (pilote Access)
Jet 4.0 a plusieurs nouvelles fonctionnalités qui ne sont pas activées par défaut dans la version 4.0 des pilotes de base de données de bureau ODBC. Pour tirer parti de ces nouvelles fonctionnalités, une application devra définir un nouvel attribut de connexion, ajouter un nouvel attribut à la chaîne de connexion ou définir l’attribut dans un DSN. L’attribut est appelé ExtendedAnsiSQL et ne concerne que les pilotes de base de données de bureau Microsoft ODBC 4.0 et plus tard.
