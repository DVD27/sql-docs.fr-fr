# <a name="sqlbindparameter-visual-foxpro-odbc-driver"></a>SQLBindParameter (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous référence de l' [API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : complète  
  
 Conformité de l’API ODBC : niveau 1  
  
 Associe une mémoire tampon à un marqueur de paramètre dans une instruction SQL. Le pilote ODBC Visual FoxPro prend en charge les paramètres d’entrée tels qu’ils sont spécifiés par l’argument *fParamType* .  
  
 Pour plus d’informations, consultez [SQLBindParameter](../../odbc/reference/syntax/sqlbindparameter-function.md) dans le *Guide de référence du programmeur ODBC*.
