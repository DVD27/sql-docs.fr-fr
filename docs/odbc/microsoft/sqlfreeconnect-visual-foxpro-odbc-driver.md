# <a name="sqlfreeconnect-visual-foxpro-odbc-driver"></a>SQLFreeConnect (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous référence de l' [API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : complète  
  
 Conformité de l’API ODBC : niveau principal  
  
 Libère un handle de connexion et libère toute la mémoire allouée pour le descripteur.  
  
 Pour plus d’informations, consultez [sqlfreeconnect,](../../odbc/reference/syntax/sqlfreeconnect-function.md) dans le *Guide de référence du programmeur ODBC*.
