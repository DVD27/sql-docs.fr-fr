# <a name="odbc-errors"></a>Erreurs ODBC
Lorsqu’une erreur se produit, les pilotes microsoft ODBC Desktop Database retournent le numéro d’erreur natif, le SQLSTATE (un code d’erreur ODBC) et un message d’erreur.
