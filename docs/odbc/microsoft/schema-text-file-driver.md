# <a name="schema-text-file-driver"></a>Schéma (pilote du fichier texte)
Le schéma comprend des informations sur chaque table (fichier texte) dans une source de données, y compris le format de la table, le nombre de lignes à numériser pour déterminer les types de colonnes, si la première rangée de la table contient des noms de colonne, si le fichier source est écrit à l’aide d’une page de code OEM ou ANSI, et le nom, le type de données et la largeur de chaque colonne.
