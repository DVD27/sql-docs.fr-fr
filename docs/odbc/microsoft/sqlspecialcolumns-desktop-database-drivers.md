# <a name="sqlspecialcolumns-desktop-database-drivers"></a>SQLSpecialColumns (pilotes pour les bases de données de poste de travail)
Un index unique sera retourné (si l’on existe) pour le drapeau SQL_BEST_ROWID en *fColType*. Aucun ensemble de résultats ne sera retourné pour le drapeau SQL_ROWVER.  
  
 Toutes les cartes d’été de ligne ont une portée de SQL_SCOPE_CURROW.  
  
 L’appariement de modèle n’est pas pris en charge pour *l’argument szTableQualifier* ou *szTableName.*
