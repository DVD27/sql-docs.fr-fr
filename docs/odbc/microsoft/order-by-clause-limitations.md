# <a name="order-by-clause-limitations"></a>ORDER BY, clause - limitations
Si une déclaration SELECT contient une clause GROUPE PAR et une clause ORDER BY, la clause ORDER BY ne peut contenir qu’une colonne dans l’ensemble de résultats ou une expression dans la clause GROUPE BY.
