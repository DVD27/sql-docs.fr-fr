# <a name="odbc-component-files"></a>Fichiers des composants ODBC
Les fichiers ODBC Driver sont requis par chaque composant des pilotes de base de données de bureau ODBC 4.0.  
  
 Certains fichiers sont requis par plus d’un composant. Ces fichiers doivent être redistribués si vous avez l’intention de redistribuer l’un des pilotes de base de données de bureau ODBC avec vos applications commerciales.  
  
 Ces fichiers sont installés dans l’annuaire Windows-System pour Windows 95 ou plus tard, ou dans l’annuaire Windows -System32 pour Windows NT 3.51 ou plus tard et Windows 2000. Si les fichiers ODBC ont déjà été installés dans un répertoire différent, assurez-vous d’utiliser les fichiers les plus récents dans l’annuaire windows -System (ou System32).
