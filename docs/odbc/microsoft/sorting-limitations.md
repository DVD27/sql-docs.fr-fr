# <a name="sorting-limitations"></a>Limitations des tris
La durée maximale d’une sorte clé dans une clause GROUPE BY, la clause ORDER BY, l’énoncé SELECT DISTINCT ou l’adhésion extérieure est de 255 octets; la longueur maximale de toutes les clés de tri dans une rangée de sorte est de 65.500 octets.
