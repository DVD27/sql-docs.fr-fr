# <a name="sorting-limitations"></a>Limitations des tris
La longueur maximale d’une clé de tri dans une clause GROUP BY, une clause ORDER BY, une instruction SELECT DISTINCT ou une jointure externe est de 255 octets ; la longueur maximale de toutes les clés de tri dans une ligne de tri est de 65 500 octets.
