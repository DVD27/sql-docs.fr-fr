# <a name="sqlgetstmtoption-desktop-database-drivers"></a>SQLGetStmtOption (pilotes pour les bases de données de poste de travail)
Les signets retournés par une *fOption* de SQL_GETBOOKMARK ne sont valables que lorsque la requête est ouverte et sont invalidées lorsque la requête est rééditée. Les signets persistants ne sont pas pris en charge.
