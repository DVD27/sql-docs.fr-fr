# <a name="product-support"></a>Prise en charge du produit
Le support produit pour ODBC est fourni par Microsoft Product Support Services. Étant donné que de nombreux produits Microsoft utilisent ODBC comme composant de base et redistribuent également les pilotes ODBC, Microsoft Product Support Services considère ODBC comme faisant partie de chaque produit qui intègre ODBC comme composant. Le soutien à l’ODBC est fourni conformément à l’accord de soutien de ce produit. Consultez la documentation du produit pour plus d’informations.
