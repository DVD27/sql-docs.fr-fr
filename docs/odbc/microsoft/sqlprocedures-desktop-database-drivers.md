# <a name="sqlprocedures-desktop-database-drivers"></a>SQLProcedures (pilotes pour les bases de données de poste de travail)
**SQLProcedures** ne retournera que les lignes pour les procédures qui ont au moins un argument. Les procédures qui n’ont pas d’arguments sont traitées comme des points de vue.  
  
|Colonne|Commentaires|  
|------------|--------------|  
|PROCEDURE_QUALIFIER|Chemin d'accès au fichier de base de données.|  
|PROCEDURE_OWNER|NULL|  
|PROCEDURE_NAME|Nom de procédure noniminé|  
|PROCEDURE_TYPE|SQL_PT_PROCEDURE|
