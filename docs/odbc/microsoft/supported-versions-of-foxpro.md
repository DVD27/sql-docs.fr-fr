# <a name="supported-versions-of-foxpro"></a>Versions de FoxPro prises en charge
Vous pouvez utiliser le pilote Microsoft® Visual FoxPro® ODBC Pour accéder aux données stockées dans les tables FoxPro. Les versions suivantes des données FoxPro sont prises en charge :  
  
-   2  
  
-   2.5  
  
-   2.6  
  
-   Visual FoxPro (toutes versions)  
  
 Lorsque vous accédez aux données stockées dans Visual FoxPro, vous pouvez choisir de vous connecter à une [base de données](../../odbc/microsoft/visual-foxpro-terminology.md) qui contient zéro ou plus de [tables](../../odbc/microsoft/visual-foxpro-terminology.md) ou à un répertoire de [tables gratuites](../../odbc/microsoft/visual-foxpro-terminology.md).  
  
 Pour plus d’informations sur la façon de se connecter à une source de données, voir [Ajouter une source de données Visuelle FoxPro](../../odbc/microsoft/adding-a-visual-foxpro-data-source.md).  
  
 Pour télécharger le pilote, consultez [le site de téléchargement Visual FoxPro ODBC Driver](https://go.microsoft.com/fwlink/?LinkId=121318).
