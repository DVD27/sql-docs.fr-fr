# <a name="native-error"></a>Erreur native
Les codes d’erreur natifs positifs sont générés par les pilotes de base de données de bureau. Les codes d’erreur négatifs sont générés par Microsoft Jet. Pour plus d’informations, consultez le *Guide du programmeur de moteurs microsoft Jet Database Engine*.
