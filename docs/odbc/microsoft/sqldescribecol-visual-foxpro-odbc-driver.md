# <a name="sqldescribecol-visual-foxpro-odbc-driver"></a>SQLDescribeCol (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous référence de l' [API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : complète  
  
 Conformité de l’API ODBC : niveau principal  
  
 Retourne le nom, le type, la précision, l’échelle et la possibilité de valeur null de la colonne de résultat donnée.  
  
 Pour plus d’informations, consultez [SQLDescribeCol](../../odbc/reference/syntax/sqldescribecol-function.md) dans le *Guide de référence du programmeur ODBC*.
