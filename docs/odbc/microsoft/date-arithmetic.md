# <a name="date-arithmetic"></a>Arithmétique des dates
Le pilote prend en charge l’ajout et la soustraction d’un entier à partir d’une colonne de DATE, d’heure ou d’horodatage. Pour une colonne de DATE, l’entier spécifie le nombre de jours à ajouter ou à soustraire. Pour une colonne d’heure ou d’horodatage, l’entier spécifie le nombre de secondes à ajouter ou à soustraire.
