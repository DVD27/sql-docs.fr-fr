# <a name="date-arithmetic"></a>Arithmétique des dates
Le conducteur prend en charge l’ajout et la soustraction d’un intégreur d’une colonne DATE, TIME ou TIMESTAMP. Pour une colonne DATE, l’integer précise le nombre de jours à ajouter ou à soustraire. Pour une colonne TIME ou TIMESTAMP, l’integer spécifie le nombre de secondes à ajouter ou à soustraire.
