# <a name="sqlputdata-visual-foxpro-odbc-driver"></a>SQLPutData (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous référence de l' [API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : complète  
  
 Conformité de l’API ODBC : niveau 1  
  
 Permet à une application d’envoyer des données pour un paramètre ou une colonne au pilote au moment de l’exécution de l’instruction.  
  
 Pour plus d’informations, consultez [SQLPutData](../../odbc/reference/syntax/sqlputdata-function.md) dans le *Guide de référence du programmeur ODBC*.
