# <a name="index-name-limitations"></a>Limitations des noms d’index
Lorsque le pilote Paradox est utilisé, un index primaire doit avoir le même nom que le tableau sur lequel il est défini. D’autres index uniques ou non uniques doivent avoir le même nom que le tableau sur lequel ils sont définis.
