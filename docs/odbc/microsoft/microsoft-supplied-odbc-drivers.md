# <a name="microsoft-supplied-odbc-drivers"></a>Pilotes ODBC fournis par Microsoft
Les *pilotes* sont des bibliothèques qui implémentent les fonctions dans l’API ODBC. Chaque pilote est spécifique à un système de gestion de base de données (SGBD) particulier. Dans la plupart des cas, les pilotes exposent les fonctionnalités du SGBD sous-jacent. ils ne sont pas requis pour implémenter des fonctionnalités non prises en charge par le SGBD. Les pilotes effectuent des tâches spécifiques et peuvent être classés en deux catégories : basées sur des fichiers et SGBD. Pour plus d’informations sur les pilotes et l’architecture des pilotes, consultez [pilotes](../../odbc/reference/drivers.md).
