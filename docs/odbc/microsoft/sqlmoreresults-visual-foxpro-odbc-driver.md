# <a name="sqlmoreresults-visual-foxpro-odbc-driver"></a>SQLMoreResults (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous référence de l' [API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : complète  
  
 Conformité de l’API ODBC : niveau 2  
  
 Détermine si davantage de résultats sont en attente sur un descripteur d’instruction, *HSTMT*, contenant des instructions SELECT, Update, INSERT ou DELETE et, le cas échéant, initialise le traitement de ces résultats.  
  
 Pour plus d’informations, consultez [SQLMoreResults](../../odbc/reference/syntax/sqlmoreresults-function.md) dans le *Guide de référence du programmeur ODBC*.
