# <a name="drop-table-statement-limitations"></a>DROP TABLE, instruction - limitations
Lorsque le pilote Microsoft Excel 5.0, 7.0 ou 97 est utilisé, l’instruction DROP TABLE efface la feuille de travail mais ne supprime pas le nom de la feuille de travail. Étant donné que le nom de la feuille de travail existe toujours dans le cahier de travail, une autre feuille de travail ne peut pas être créée du même nom.
