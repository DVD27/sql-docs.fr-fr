# <a name="sqlparamoptions-mapping"></a>SQLParamOptions, mappage
Lorsqu’une application appelle **SQLParamOptions** par l’intermédiaire d’un conducteur *ODBC 3.x,* l’appel  
  
```  
SQLParamOptions(hstmt, crow, piRow);  
```  
  
 sera cartographié à deux appels de **SQLSetStmtAttr** comme suit:  
  
```  
SQLSetStmtAttr(hstmt, SQL_ATTR_PARAMSET_SIZE, crow, 0);  
SQLSetStmtAttr(hstmt, SQL_ATTR_PARAMS_PROCESSED_PTR, piRow, 0);  
```
