# <a name="odbc-header-files"></a>Fichiers d’en-tête ODBC
> [!NOTE]  
>  La redistribution des fichiers d’en-tête ODBC n’est pas nécessaire. Aucun n’est considéré comme « Core ».  
  
 **Fichiers d’en-tête**  
  
 Odbcinst. h  
  
 SQL. h  
  
 Sqlext. h  
  
 SqlTypes. h  
  
 Sqlucode. h  
  
 MSDASQL. h  
  
 Msdadc. h  
  
 Sqlspi. h (utilisé pour le développement du pilote, consultez Référence de l' [interface SPI (Service Provider Interface) ODBC](../../../odbc/reference/syntax/odbc-service-provider-interface-spi-reference.md) pour plus d’informations) disponible à partir du kit de développement logiciel (SDK) Windows 8.
