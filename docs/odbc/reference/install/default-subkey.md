# <a name="default-subkey"></a>Sous-clé par défaut
Le registre peut spécifier une source de données par défaut avec le sous-clé par défaut. Ce sous-clé est un cas particulier d’un sous-clé de spécification de source de données et a les mêmes valeurs que n’importe quel autre sous-clé de spécification de source de données. La seule différence est qu’il n’est pas répertorié comme une valeur sous le sous-clé ODBC Data Sources.
