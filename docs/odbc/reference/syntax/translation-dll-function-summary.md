# <a name="translation-dll-function-summary"></a>Récapitulatif des fonctions de la DLL de traduction
Le tableau suivant décrit la traduction des fonctions DLL. Pour plus d’informations sur la syntaxe et la sémantique pour chaque fonction, voir [Traduction DLL Fonction Référence](../../../odbc/reference/syntax/translation-dll-api-reference.md).  
  
|Tâche|Nom de la fonction|Objectif|  
|----------|-------------------|-------------|  
|Traduire les données|[SQLDataSourceToDriver](../../../odbc/reference/syntax/sqldatasourcetodriver-function.md)|Traduit toutes les données qui circulent de la source de données au conducteur.|  
||[SQLDriverToDataSource (en anglais)](../../../odbc/reference/syntax/sqldrivertodatasource-function.md)|Traduit toutes les données qui circulent du conducteur vers la source de données.|
