# <a name="sqlsetparam-function"></a>SQLSetParam, fonction
**Conformité**  
 Version introduite: ODBC 1.0 Standards Compliance: Deprecated  
  
 **Résumé**  
 Dans ODBC 2.0, la fonction ODBC 1.0 **SQLSetParam** a été remplacée par **SQLBindParameter**. Pour plus d’informations, voir [SQLBindParameter Function](../../../odbc/reference/syntax/sqlbindparameter-function.md).  
  
> [!NOTE]  
>  Pour plus d’informations sur ce que le Driver Manager cartographie cette fonction à quand un ODBC 2. *x* application fonctionne avec un ODBC 3. *x* pilote, voir [Mapping Deprecated Functions](../../../odbc/reference/appendixes/mapping-deprecated-functions.md).  
  
## <a name="remarks"></a>Notes  
 Consultez [ODBC 64-Bit Information](../../../odbc/reference/odbc-64-bit-information.md), si votre application s’exécute sur un système d’exploitation 64 bits.
