# <a name="sqlsetparam-function"></a>SQLSetParam, fonction
**Conformité**  
 Version introduite : conformité des normes ODBC 1,0 : déconseillé  
  
 **Résumé**  
 Dans ODBC 2,0, la fonction ODBC 1,0 **SQLSetParam,** a été remplacée par **SQLBindParameter**. Pour plus d’informations, consultez [SQLBindParameter, fonction](../../../odbc/reference/syntax/sqlbindparameter-function.md).  
  
> [!NOTE]  
>  Pour plus d’informations sur le mappage de cette fonction par le gestionnaire de pilotes lorsqu’un ODBC 2. l’application *x* fonctionne avec ODBC 3. *x* , consultez [mappage des fonctions dépréciées](../../../odbc/reference/appendixes/mapping-deprecated-functions.md).  
  
## <a name="remarks"></a>Notes  
 Consultez [ODBC 64-informations sur ODBC](../../../odbc/reference/odbc-64-bit-information.md), si votre application s’exécute sur un système d’exploitation 64 bits.
