# <a name="diagnostic-tools"></a>Outils de diagnostic
Deux fonctionnalités peuvent vous aider à diagnostiquer les problèmes dans les applications ODBC. Les deux sont disponibles auprès de l’administrateur ODBC et implémentées par le gestionnaire de pilotes. Le *suivi* fournit un moyen d’enregistrer dans un fichier journal la séquence d’appels de fonction. Visual Studio Analyzer permet d’analyser des informations sur l’interaction des composants dans un environnement distribué.  
  
 Cette section contient les rubriques suivantes :  
  
-   [Suivi](../../../odbc/reference/develop-app/tracing.md)  
  
-   [Visual Studio Analyzer](../../../odbc/reference/develop-app/visual-studio-analyzer.md)
