# <a name="diagnostic-tools"></a>Outils de diagnostic
Deux installations peuvent aider à diagnostiquer les problèmes dans les applications ODBC. Les deux sont disponibles auprès de l’administrateur de l’ODBC et mis en œuvre par le gestionnaire de conducteur. *Le traçage* fournit un moyen d’enregistrer à un fichier journal la séquence des appels de fonction. Visual Studio Analyzer permet l’analyse d’informations sur l’interaction des composants dans un environnement distribué.  
  
 Cette section contient les rubriques suivantes :  
  
-   [Traçage](../../../odbc/reference/develop-app/tracing.md)  
  
-   [Visual Studio Analyzer](../../../odbc/reference/develop-app/visual-studio-analyzer.md)
