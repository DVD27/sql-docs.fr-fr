# <a name="batch-processing"></a>Traitement par lots
La prise en charge des lots d’instructions SQL n’étant pas répandue, les applications interopérables doivent les utiliser de manière conditionnelle ou pas du tout. Pour plus d’informations, consultez [exécution de lots](../../../odbc/reference/develop-app/executing-batches.md).
