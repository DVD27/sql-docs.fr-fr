# <a name="tables-and-views"></a>Tables et vues
Dans les fonctions ODBC, les tables et les vues sont interchangeables. Le *tableau* de terme est utilisé à la fois pour les tables et pour les vues, sauf lorsque la *vue* de terme est utilisée explicitement.
