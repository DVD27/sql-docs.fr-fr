# <a name="tracing"></a>Traçage
Le gestionnaire de conducteur de l’ODBC dispose d’une installation de traçabilité qui permet d’enregistrer et de transcrire la séquence d’appels de fonction effectués par une application ODBC dans un fichier journal. Le traçage est effectué par une trace DLL qui capture les appels entre l’application et le gestionnaire de conducteur, et entre le gestionnaire de conducteur et le conducteur. Cette méthode de traçage remplace le tracé effectué par l’ODBC 2 *.x* Driver Manager et le tracé effectué dans ODBC 2 *.x* par ODBC Spy.
