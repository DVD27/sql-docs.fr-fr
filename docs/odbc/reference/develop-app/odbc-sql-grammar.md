# <a name="odbc-sql-grammar"></a>Grammaire SQL d’ODBC
Les applications interopérables doivent toujours utiliser la grammaire SQL ODBC dans les relevés SQL. Cependant, des variations considérables sont possibles même dans cette grammaire. Pour plus d’informations, voir [Interopérabilité des déclarations SQL](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md).
