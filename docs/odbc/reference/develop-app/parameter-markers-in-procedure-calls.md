# <a name="parameter-markers-in-procedure-calls"></a>Marqueurs de paramètre dans les appels de procédure
Lors de l’appel des procédures qui acceptent des paramètres, les applications interopérables doivent utiliser des marqueurs de paramètres au lieu de valeurs de paramètres littérales. Certaines sources de données ne prennent pas en charge l’utilisation des valeurs de paramètres littérales dans les appels de procédure. Pour plus d’informations sur les paramètres, voir [Paramètres d’instruction](../../../odbc/reference/develop-app/statement-parameters.md). Pour plus d’informations sur les procédures d’appel, voir [Les appels de procédure](../../../odbc/reference/develop-app/procedure-calls.md), plus tard dans cette section.
