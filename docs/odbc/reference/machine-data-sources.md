# <a name="machine-data-sources"></a>Sources de données de machine
*Les sources de données de* machine sont stockées sur le système avec un nom défini par l’utilisateur. Associé au nom de la source de données est toute l’information dont le gestionnaire de conducteur et le conducteur ont besoin pour se connecter à la source de données. Pour une source de données Xbase, il peut s’agir du nom du pilote Xbase, de la trajectoire complète de l’annuaire contenant les fichiers Xbase, et de certaines options qui indiquent au pilote comment utiliser ces fichiers, tels que le mode utilisateur unique ou la lecture seulement. Pour une source de données Oracle, il peut s’agir du nom du pilote Oracle, du serveur où réside\*l’Oracle DBMS, de la chaîne de connexion SQL-Net qui identifie le pilote SQL Net à utiliser et de l’ID système de la base de données sur le serveur.
