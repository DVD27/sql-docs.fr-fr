# <a name="procedures-collection-adox"></a>Procedures, collection (ADOX)
Contient tous les objets de [procédure](../../../ado/reference/adox-api/procedure-object-adox.md) d’un catalogue.  
  
## <a name="remarks"></a>Notes  
 La méthode [Append](../../../ado/reference/adox-api/append-method-adox-procedures.md) d’une collection **procedures** est unique pour ADOX. Vous pouvez :  
  
-   Ajoutez une nouvelle procédure à la collection à l’aide de la méthode **Append** .  
  
 Les propriétés et les méthodes restantes sont standard pour les collections ADO. Vous pouvez :  
  
-   Accédez à une procédure dans la collection à l’aide de la propriété [Item](../../../ado/reference/ado-api/item-property-ado.md) .  
  
-   Retourne le nombre de procédures contenues dans la collection avec la propriété [Count](../../../ado/reference/ado-api/count-property-ado.md) .  
  
-   Supprimez une procédure de la collection à l’aide de la méthode [Delete](../../../ado/reference/adox-api/delete-method-adox-collections.md) .  
  
-   Mettez à jour les objets de la collection pour refléter le schéma de base de données actuel avec la méthode [Refresh](../../../ado/reference/ado-api/refresh-method-ado.md) .
