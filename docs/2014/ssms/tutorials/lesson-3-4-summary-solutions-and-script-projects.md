# <a name="summary-solutions-and-script-projects"></a>Résumé : Solutions et projets de script
  En tirant parti des fonctionnalités de développement étendues de Visual [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] Studio, offre les fonctionnalités dont les développeurs ont besoin pour la gestion du [!INCLUDE[msCoName](../../includes/msconame-md.md)] code au niveau de l’entreprise. Vous pouvez créer des modèles personnalisés qui prennent en compte l'aspect répétitif de vos tâches, enregistrer et gérer des projets de scripts complexes dans un environnement centralisé et coordonné.
