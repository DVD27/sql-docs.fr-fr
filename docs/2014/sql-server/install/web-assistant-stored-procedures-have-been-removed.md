# <a name="web-assistant-stored-procedures-have-been-removed"></a>Des procédures stockées de l'Assistant Web ont été supprimées
  Le Conseiller de mise à niveau a détecté l'utilisation de procédures stockées de l'Assistant Web.  
  
## <a name="component"></a>Composant  
 [!INCLUDE[ssDE](../../includes/ssde-md.md)]  
  
## <a name="description"></a>Description  
 Les procédures stockées de l'Assistant Web **sp_makewebtask**, **sp_dropwebtask**, **sp_runwebtask**et **sp_enumcodepages** ont été supprimées.  
  
## <a name="corrective-action"></a>Action corrective  
 Nous vous recommandons d'utiliser [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] à la place.
