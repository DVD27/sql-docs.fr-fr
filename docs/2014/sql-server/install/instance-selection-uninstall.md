# <a name="instance-selection-uninstall"></a>Sélection de l'instance (Désinstaller)
  Dans la page sélection d’instance-désinstaller, utilisez la zone de liste déroulante pour [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] spécifier l’instance de à désinstaller.  
  
## <a name="options"></a>Options  
 Vous pouvez spécifier une instance entière de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]ou bien indiquer que seuls les fonctionnalités partagées et les outils d'administration doivent être désinstallés.
