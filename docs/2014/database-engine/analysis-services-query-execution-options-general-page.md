# <a name="analysis-services-query-execution-options-general-page"></a>Options d'exécution de la requête d'Analysis Services (page Général)
  Utilisez cette page pour spécifier les options d'affichage des résultats d'une requête [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] .  
  
 **Délai d’exécution**  
 Spécifie le nombre de secondes à attendre avant d'annuler la requête. La valeur 0 indique un délai d'attente illimité ou pas de délai.  
  
 **Rétablir les valeurs par défaut**  
 Rétablit les valeurs d'origine par défaut de toutes les options de cette page.
