## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|SQL Server|  
|ID de l’événement|256|  
|Source de l’événement|Runtime de base de données locale SQL Server 12.0|  
|Composant|API d'exécution de la base de données locale|  
|Texte du message|Impossible de créer un dossier pour l’instance de base de données locale dans :%% LOCALAPPDATA%%\\ \ Microsoft\Microsoft SQL Server\>DB\Instances local<nom de l’instance.|
