# <a name="iswebserviceenabled-property-wmi-msreportserver_configurationsetting"></a>Propriété IsWebServiceEnabled (WMI MSReportServer_ConfigurationSetting)
  Indique si le service Web Report Server est activé. Lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim IsWebServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWebServiceEnabled;  
```  
  
## <a name="property-values"></a>Valeurs de la propriété  
 Valeur `Boolean` en lecture seule. La valeur `true` indique que le service Web Report Server est activé.
