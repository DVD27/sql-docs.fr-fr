# <a name="raw-file-source-editor-columns-page"></a>Éditeur de source de fichier brut (page Colonnes)
  La source de fichier brut lit des données brutes dans un fichier. La représentation des données étant native pour la source, celles-ci ne nécessitent aucune traduction et pratiquement aucune analyse. Pour plus d’informations, consultez [Source de fichier brut](data-flow/raw-file-source.md).
