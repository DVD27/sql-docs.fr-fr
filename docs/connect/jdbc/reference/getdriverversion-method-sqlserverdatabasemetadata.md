# <a name="getdriverversion-method-sqlserverdatabasemetadata"></a>Méthode getDriverVersion (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère le numéro de version de ce pilote JDBC.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public java.lang.String getDriverVersion()  
```  
  
## <a name="return-value"></a>Valeur de retour  
 **String** contenant la version du pilote JDBC.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode getDriverVersion est spécifiée par la méthode getDriverVersion de l’interface java.sql.DatabaseMetaData.
