# <a name="clearwarnings-method-sqlserverresultset"></a>Méthode clearWarnings (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Efface tous les avertissements signalés sur cet objet [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md).  
  
> [!NOTE]  
>  Actuellement, cette méthode n’est pas implémentée par [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]. Si elle est appelée, elle renvoie toujours la valeur Null.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public void clearWarnings()  
```  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode clearWarnings est spécifiée par la méthode clearWarnings de l’interface java.sql.ResultSet.
