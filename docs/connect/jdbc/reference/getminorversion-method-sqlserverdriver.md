# <a name="getminorversion-method-sqlserverdriver"></a>Méthode getMinorVersion (SQLServerDriver)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retourne le numéro de la version mineure de [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)].  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public int getMinorVersion()  
```  
  
## <a name="return-value"></a>Valeur de retour  
 **int** contenant le numéro de version mineure du pilote JDBC.  
  
## <a name="remarks"></a>Notes  
 Cette méthode getMinorVersion est spécifiée par la méthode getMinorVersion de l’interface java.sql.Driver.
