# <a name="getdatabaseproductname-method-sqlserverdatabasemetadata"></a>Méthode getDatabaseProductName (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère le nom de ce produit de base de données.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public java.lang.String getDatabaseProductName()  
```  
  
## <a name="return-value"></a>Valeur de retour  
 **Chaîne** contenant le nom de produit de la base de données.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode getDatabaseProductName est spécifiée par la méthode getDatabaseProductName de l’interface java.sql.DatabaseMetaData.
