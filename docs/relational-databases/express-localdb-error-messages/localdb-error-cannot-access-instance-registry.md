[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
    
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|SQL Server|  
|ID de l’événement|262|  
|Source de l’événement|Runtime de base de données locale SQL Server 12.0|  
|Composant|API d'exécution de la base de données locale|  
|Texte du message|Une erreur inattendue s'est produite lors de la tentative d'accès à la configuration du Registre pour l'instance de base de données locale. Pour plus d'informations sur l'erreur, consultez le journal des applications Windows.|  
  
## <a name="explanation"></a>Explication  
 Un Registre d'instance n'est pas accessible.
