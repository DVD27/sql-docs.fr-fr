# <a name="flagindex-property-clientsettingsgeneralflag-class"></a>Propriété FlagIndex (classe ClientSettingsGeneralFlag)
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sqlserver.md)]
  Obtient la valeur d'index de l'indicateur.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
object.FlagIndex [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *object*  
 Objet de [classe ClientSettingsGeneralFlag](../../../relational-databases/wmi-provider-configuration-classes/clientsettingsgeneralflag-class/clientsettingsgeneralflag-class.md) qui représente un indicateur général pour les paramètres du serveur.  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur u**Int32** qui spécifie la valeur d’index de l’indicateur.
