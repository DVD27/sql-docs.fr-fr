# <a name="working-with-user-defined-types-in-sql-server"></a>Utilisation de types définis par l'utilisateur dans SQL Server
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  Vous pouvez accéder à la fonctionnalité de type [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] (UDT) définie par l’utilisateur à partir de la langue en utilisant la [!INCLUDE[tsql](../../includes/tsql-md.md)] syntaxe de requête régulière. Les types UDT peuvent être utilisés dans la définition d'objets de base de données, tels que les variables dans les lots [!INCLUDE[tsql](../../includes/tsql-md.md)], dans les fonctions et les procédures stockées, et comme arguments dans les fonctions et les procédures stockées.
