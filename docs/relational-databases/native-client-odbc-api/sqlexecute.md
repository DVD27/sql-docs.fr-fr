# <a name="sqlexecute"></a>SQLExecute
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  Si l’attribut de déclaration SQL_SOPT_SS_PARAM_FOCUS n’est pas réglé à 0, SQLExecute retournera SQL_ERROR et générera un dossier diagnostique avec SQLSTATE-HY024 et le message " Valeur d’attribut invalide, SQL_SOPT_SS_PARAM_FOCUS (doit être zéro au moment de l’exécution)". Pour plus d'informations sur SQL_SOPT_SS_PARAM_FOCUS, consultez [SQLSetStmtAttr](../../relational-databases/native-client-odbc-api/sqlsetstmtattr.md).
