# <a name="rename-a-database-diagram-visual-database-tools"></a>Renommer un diagramme de base de données (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
Vous pouvez renommer un diagramme de base de données dans l’Explorateur d’objets. Nécessite l'autorisation ALTER sur la base de données.  
  
### <a name="to-rename-a-database-diagram"></a>Pour renommer un diagramme de base de données  
  
1.  Dans l’Explorateur d’objets, développez le dossier **Diagrammes de base de données**.  
  
2.  Cliquez avec le bouton droit sur le diagramme de base de données que vous souhaitez renommer.  
  
3.  Dans le menu contextuel, sélectionnez **Renommer** .  
  
    Une zone d'édition s'ouvre autour du nom du schéma dans l'Explorateur d'objets.  
  
4.  Entrez le nouveau nom du diagramme de base de données et appuyez sur **Entrée**.  
  
    Le diagramme de base de données apparaît sous son nouveau nom dans l’Explorateur d’objets.
